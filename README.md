# meme-generator
<h1>Python Wholesome Meme Generator</h1>

<h2>Stack</h2>
- Python
- Flask
- API that handles meme generation (credit to: https://meme-api.com/gimme)

<h2>What you need</h2>
- Python installed

- Flask installed

- virtual environment depending on your machine

<h2>Troubleshooting</h2>
Server error?
- Try deleting browser cookies.

- Still not working? Try flushing sockets (chrome://net-internals/#sockets)

<h2>How to run the app</h2>
- Navigate to the directory where the file is stored. Make sure to accesss the folder that has the meme_flask.py file in it.
- Once you've navigated to the correct file, in the terminal, input the command:
```
flask --app meme_flask.py run
```
- Once the server shows in the terminal, command + click the link to navigate to the website in the browser.
